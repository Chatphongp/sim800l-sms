# Node SMS Reader
Read incoming SMSs from SIM800L Module connected with Arnuino via serial port.

## Hardwares needed
    - Arduino Uno R3
    - Sim 800L Module
    - Power Supply (at least 5V3A). Arduino own supply is not enough.
